FACEMETA - Hominological Dataset of Facial Image Metadata

FACEMETA is the largest commercially available dataset of facial images with detailed metadata.

TThe FACEMETA dataset is intended for use in academic research and corporate research and development (R&D). It contains over 50,000 normalized photographs of male and female faces of varying ethnicity between the ages of 17-65. Extensive data processing are made available for each individual model. This dataset includes physical attributes as well facial landmarks and facial feature encodings for each individual model.

Overview

The metadata for face images include descriptors such as: ethnicity, gender, age, weight, height, eye color, and date picture taken. Additionally, pre-processing techniques were applied to accelerate deep learning projects, including standardization of aspect ratio, image size, image alignment, tilt, noise reduction, standardized field of view, application of 64 feature points, and image encoding.

This dataset’s images are pre-processed and ready for modeling. Use cases include:

  • Age/Health prediction
  • ID matching
  • Psychology studies
  • Facial expression analysis
  • Face generation
  • Sexual orientation
  • Life span

    • Methodology

    Face images were acquired in real setting. We first used the Histogram of Oriented Gradients method to detect the face in each image [1]. Then, 68 facial landmark were extracted using the face alignment method by Kazemi and Josephine [2]. An affine image transformation were then apply to map landmark for each faces. Each image is also embedded into a 128 dimensional vector [3].

    About

    We are a collective of data scientists researching computer vision within psychology, sociology, behavioural economics, and other research fields. FACEMETA addresses common challenges we’ve faced in procuring and preparing data for analysis.
