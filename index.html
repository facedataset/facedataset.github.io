FACEMETA - Hominological Face Dataset With Image Metadata

FACEMETA is the largest commercially available dataset of facial images with detailed metadata. The FACEMETA dataset is intended for use in academic research and corporate R&D. It contains 100,000 normalized photographs of male and female faces of varying ethnicity between the ages of 18 and 80. The FACEMETA dataset includes physical attributes as well as facial landmarks and facial feature encodings for each individual image.

Overview

The FACEMETA dataset includes normalized images and the following metadata and features: gender, age, ethnicity, height, weight, 68 facial landmarks, and a 128-dimensional embedding for each normalized images. Multiple pre-processing techniques were applied to obtain the normalized images. Those include normalizing image size, aspect ratio and field of view, reducing image noise, and correcting for various head pose. This pre-processing significantly speeds up the data preparation step for your deep learning projects that make use of face images.

The FACEMETA dataset is ready to be used within your algorithm development process. Several use cases include:

  • Age prediction
  • ID matching
  • Psychology studies
  • Facial expression analysis
  • Face generation
  • Sexual orientation
  • Life span estimation

    • Methodology

    Face images were acquired in real setting. The Histogram of Oriented Gradients Method by Dalal and Triggs was used to detect the face in each image [1]. Then, 68 facial landmarks were extracted using the face alignment method as presented by Kazemi and Josephine [2]. Those landmarks were used to parameterize an affine image transformation to normalize the images into a common space. Finally, 68 facial landmarks were extracted from the normalized images and a 128-dimensional embedding is computed for each normalized image using the method presented by Schroff et al [3].

    About

    We are a collective of independent data scientists and researchers across various disciplines including psychology, sociology, and behavioural economics. FACEMETA addresses common challenges faced in procuring and processing data for analysis.
